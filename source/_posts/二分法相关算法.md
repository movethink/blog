## 二分查找

二分查找主要点在于边界条件判断，即结束 while 循环的时机。

当判断左边界大于右边界时，即可确定数组内没有要查找的元素，即可跳出循环体。

```js
/**
* nums 检索数组
* target 检索值
*/
search(nums, target) {
  let left = 0,
    right = nums.length - 1;
  while (left <= right) {
    let middle = Math.floor((left + right) / 2);
    if (nums[middle] > target) {
      right = middle - 1;
    } else if (nums[middle] < target) {
      left = middle + 1;
    } else {
      return middle;
    }
  }
  return -1;
}
```
